# TodoList

#### Description
一个非常简单的todoList小项目，支持云端同步功能(需要注册账号)，前端使用React框架和BootStrap库构建，后端使用SSM框架连接MySQL数据库，前后端分离的小项目，并使用了Java设计模式优化代码。
